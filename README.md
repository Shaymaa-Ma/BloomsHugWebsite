# BloomsHugWebsite

A full-stack flowershop website built with HTML, CSS, JavaScript, PHP, and MySQL.

## Features
- Browse products by category
- Add to cart and checkout
- User registration and login
- Contact form
- Responsive design

## Tech Stack
- HTML, CSS, JavaScript
- PHP
- MySQL
- WAMP/XAMPP for local development

## How to Run Locally
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/BloomsHugWebsite.git`
2. Copy the folder to `C:\wamp64\www\`
3. Import `blooms.sql` into phpMyAdmin
4. Open `http://localhost/BloomsHugWebsite` in your browser
